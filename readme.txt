Description
----
Express is a Free Responsive Magazine WordPress Theme with quiet simply design, your readers would have the friendly feeling which helps them to focus on “reading” content than “looking” your blog.

Features list
----
 - High Speed
 - Social Media Integration
 - Fully Responsive & Retina Ready
 - Search Engines Optimized
 - Easy Customizable
 - Build with Bootstrap
 - Supports FontAwesome
 - Write on HTML5 / LESS
 - Translation Ready
